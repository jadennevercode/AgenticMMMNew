# 2.24 数据集 / Model Data（模版 / 输出 Schema）

> 类别：Output ｜ 所有数据源按统一维度集成后的**数据全集**。出处：`Data Process_2.24` + `2.21&2.23 / Model Data`。

## 数据集表（`vw_all_dataset_model_data_yyyymm`）列定义

| 列 | 类型 | 说明 |
| --- | --- | --- |
| TASKNAME | String | 来源 task 名（对应 2.22 TaskLog） |
| 品牌 | String | 模型颗粒度 |
| 省份组别 | String | 模型颗粒度（本案大区：KA/华东/华西/华南/华北） |
| 省份 | String | 模型颗粒度 |
| 渠道类型 | String | 模型颗粒度 |
| 渠道 | String | 模型颗粒度 |
| 年 / 月 | String | 时间颗粒度 |
| 数据源 | String | 来源标识 |
| 数据类型Level1–Level4 | String | 因子树 L1–L4 |
| 数据类型Level5–Level8 | String | 下钻维度（无则 `NA`） |
| METRICS类型 | String | 指标类型（如 value / Impression / spending / Volume） |
| METRICS | String | 指标名（如 sales value / 铁路客运量） |
| VALUE | String/Number | 值 |
| Variable no. | String | 建模变量编号 |
| Metric no. | String | 建模指标编号 |

## 与下游关系

- **2.24 = 全集**（所有通过验收的源，含未必入模的指标）。
- **2.35 Model Input = 子集**（经 2.33 技术检验 + 2.34 OLS 筛选后，加上 `Variable no. / Metric no.` 的最终入模指标）。
