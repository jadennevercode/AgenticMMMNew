# 2.33 指标技术校验规则（模版 / 技术统计检验）

> 类别：Input ｜ 技术校验：看指标是否合建模标准。一个因子有多个指标时，部分会在此被筛掉；
> 若某因子仅有的指标**全部**不达标 → **预警**。出处：`Data statistical test_2.33`。

## 三项检验 + 打分

### 1. 波动性检验（CV）
波动系数 CV：先将数据缩放到 0–1，再算 `方差 / 均值`。

| 分 | 条件 | 含义 |
| --- | --- | --- |
| 0 | CV ≤ 0.05 | 变异度极小，基本无波动，**强烈建议剔除**或找替代 |
| 0.5 | 0.05 < CV < 0.1 | 变异度弱，除非业务不可替代否则不建议入模 |
| 1 | 0.1 ≤ CV < 0.2 | 变异度中等，波动较明显 |
| 2 | CV ≥ 0.2 | 变异度高，波动明显 |

### 2. 相关性检验（Pearson r，因子 vs KPI，单变量）

| 分 | 条件 | 含义 |
| --- | --- | --- |
| 0 | \|r\| < 0.1 | 基本不相关，不建议入模 |
| 0.5 | 0.1 ≤ \|r\| < 0.3 | 弱相关，因子多且业务弱可忽略，因子少/重要则入模 |
| 1 | 0.3 ≤ \|r\| < 0.5 | 较相关，建议入模 |
| 2 | \|r\| ≥ 0.5 | 强相关，建议入模 |

### 3. 共线性检验（VIF，因子 vs 因子）

| 分 | 条件 | 含义 |
| --- | --- | --- |
| 0 | VIF = 1 | 与其他特征无线性相关 |
| 0.5 | 1 < VIF < 5 | 中等共线性，一般可接受 |
| 1 | VIF ≥ 5 | 明显共线性，需关注 |
| 2 | VIF ≥ 10 | 严重共线性，通常必须处理 |

> 注：VIF 越大共线性越严重；表中"分值"是按需关注程度标注，处理时应结合业务取舍（高 VIF 指标需降维/合并/剔除）。

## 合成判定

```
Total = CV + Pearson + VIF
  Good          if  Total ≥ 3
  Acceptable    if  1.5 ≤ Total < 3
  unconsiderable if Total < 1.5
```

## 与 2.1 的区别（重要）

| | 2.1 数据校验 | 2.33 技术校验 |
| --- | --- | --- |
| 衡量对象 | 数据**本身质量** | 指标对**建模的适用性** |
| 维度 | 一致性/准确性/完整性/颗粒度 | CV / Pearson / VIF |
| 失败处置 | 0 不能用、0.5 人决策 | 不达标淘汰指标；因子全失 → 预警 |
