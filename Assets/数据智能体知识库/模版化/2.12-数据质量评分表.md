# 2.12 数据质量评分表（模版 / 输出 Schema）

> 类别：Output ｜ 数据校验的产出物：逐指标的质量评分卡。出处：`Data Validation_2.1 / 2.12数据质量评分`。

## 表头结构（每行 = 一个 L4 因子下的一个指标）

| 列 | 含义 | 取值 |
| --- | --- | --- |
| L1 / L2 / L3 / L4 | 因子树四级 | 文本（来自因子树） |
| 指标 | 该因子下的具体指标 | 如 曝光量 / 花费 |
| 一致性评分 | Consistency 得分 | 符合=1，其余=0（或 0.5） |
| 一致性情况 | 文字说明 | 按校验标准总结 |
| 完整性评分 / 完整性情况 | Comprehensiveness | 同上 + 文字（如 `2 years+`） |
| 颗粒度评分 / 颗粒度情况 | Granularity | 同上 + 文字（如 `monthly`） |
| Brand Granularity Compliancy | 品牌颗粒度合规 | category / brand / … |
| Granularity at which the data exists | 数据实际存在的颗粒度 | 文本 |
| Granularity collected | 采集到的颗粒度 | 文本 |
| Brand Granularity integrity | 品牌颗粒度完整性 | complete / incomplete / missing |
| Channel Granularity Compliancy / exists / collected / integrity | 渠道颗粒度系列 | 同品牌列 |
| 真实性评分 / 真实性情况 | Accuracy | 符合=1，其余=0 + 文字 |
| Total Score | 合成总分 | 见下 |

## 验收标准（数据验收口径）

```
1. Consistency  : 数据计算逻辑未变 =1，变化 =0
2. Frequency    : 周度/月度 =1，年度 =0
3. Data coverage: 周期 >2年 =1，1–2年 =0.5，<1年 =0
4. Brand integrity   : complete=1 / incomplete=0.5 / missing=0
5. Channel integrity : complete=1 / incomplete=0.5 / missing=0
6. Geography integrity: complete=1 / incomplete=0.5 / missing=0

Final score = 完整性 + 颗粒度 + 真实性 + 一致性
  score = 1   → 验收
  score = 0   → 数据完全不能用
  score = 0.5 → 人进去决策如何处理
```
