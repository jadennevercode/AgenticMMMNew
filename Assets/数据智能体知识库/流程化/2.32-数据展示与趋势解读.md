# 2.32 数据展示与趋势解读（流程化 / 输出）

> 类别：Output ｜ 业务交叉校验（2.31）的产出形态：Charting + 数据趋势解读 + 需业务校验的问题。
> 出处：`Data Analysis_2.32 Charting.xlsx` + `数据趋势解读.pptx`。

## 产出三件套

1. **Charting**：基于 `2.24 数据集` 做多维透视图。
2. **数据趋势解读**：每张图配业务解读（增长/下滑/突变点/季节规律/跑赢跑输大盘）。
3. **需与客户确认的数据问题清单**：异常点 + 假设 + 待确认项。

## Charting 维度（PIVOT 结构）

工作簿中的透视分析按多维展开（对应 2.31 六步的"维度拆解"）：

| 透视 | 拆解维度 | 用途 |
| --- | --- | --- |
| Summary TOTAL | MKT / SALES / E-commerce 花费汇总（by year/H1） | 全景概览 |
| PIVOT by Region | 大区 × 指标 | 区域拆解 |
| PIVOT 店类型 | 渠道类型 × 指标 | 渠道拆解 |
| PIVOT 产品 / 产品CPR | 品牌 × 产品 × KPI（RSP/Value/Volume）| 产品拆解 |
| MODEL Design | base / internal / external / marketing 因子归类 | 建模设计对齐 |
| KPI / Base / PFME / Promotion / Finance | 按因子大类的指标透视 | 分模块解读 |

## 解读规则（步骤1 全景概览示例）

| 看什么 | 图表 | 解读要点 | 结论示例 | 相关因子(L4) |
| --- | --- | --- | --- | --- |
| 整体销量/销售额趋势 | 月度折线(3年) | 增长还是下降？突变点？季节性？ | "2025YTD +3%，8 月骤降 12%" | 市场规模 |
| 同比增速 | 年度/季度柱状 | 是否跑赢大盘？ | "销量 +3%，品类 +5%，跑输" | 市场规模 |
| 市场份额变化 | 月度份额曲线 | 份额升还是降？ | "30.5%→29%" | 本品ND、竞品ND |
| 费用 vs 销量叠图 | 双轴图 | 费用涨是否带来销量涨？ | "费用 +10% 销量仅 +3%" | 各类花费 |
| 品牌搜索量趋势 | 搜索指数折线 | 领先指标健康吗？ | "搜索 -15% 预示销量压力" | 品类社媒声量 |

## 输出示例（端到端）

| 步骤 | 输出结论 | 涉及因子 |
| --- | --- | --- |
| 全景概览 | 销量 +3% 但份额 30.5%→29%，跑输品类(+5%)，搜索 -15% | 市场规模, 本品ND, 品类社媒声量 |
| 渠道拆解 | 抖音 ROI 1.2，天猫 2.0，线下 0.6；TT/AFH 存在虚增(已标注) | 各渠道花费, 本品ND |
| 异常定位 | 8 月 抖音×头部达人×产品演示内容 ROI -43% | 按 campaign 下钻 |
| 先验提取 | 抖音弹性先验设 Normal(0.45,0.1) 截断 [0.3,0.7] | 抖音花费 |
| 客户核对 | 客户确认 8 月代理更换、内容质量下降，同意弹性上限 0.7 | — |

> Charting 是**人审 + 客户对齐**的载体，最终把"业务确认后的假设"固化为模型先验，喂给建模阶段。
