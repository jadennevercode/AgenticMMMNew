## 数据智能体

| 编号 | 类别 | 文件 | 任务 |

| 2.1 | Input/Output | 【MMM AI】数据智能体-Data Validation | |
| 2.11 | Input | 数据通用校验标准 | 通用标准 + 打分规则 |
| 2.12 | Output | 数据质量评分 | 质量总结 |
| 2.2 | Input/Output | 【MMM AI】数据智能体-Data Process | |
| 2.21 | Input | 宽表数据集维度 | 根据模型颗粒度 + 时间颗粒度 + 因子 L1-L4 + 因子 Deepdive L5-L8 + 指标类型定义 |

| 2.31 | Input | 指标业务校验规则 | 通过数据总结趋势及数据交叉逻辑，并产出需要与客户确认的数据问题 |
| 2.32 | Output | 数据展示 | Charting + 数据趋势解读 + 业务需要校验的问题 |
